# MERN Product Management App 🛒

A **Project** built to improve my **MERN stack skills**.  
This app allows users to manage products with **full CRUD operations**, built with **React, Node.js, Express, and MongoDB**, featuring **Chakra UI**, **responsive design**, and **dark/light theme support**.

---

## 🚀 Live Demo
Check out the deployed app here: [Product-App](https://mern-product-app-1-75cs.onrender.com)

---

## 🛠 Tech Stack

- **Frontend:** React, Chakra UI, HTML5, CSS3, JavaScript  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Tools:** npm, Postman, Git/GitHub  
- **Deployment:** Render  

---

## ⚙️ Features Practiced

- **CRUD operations:** Add, Edit, Delete, and View products  
- **Responsive design:** Works on mobile and desktop  
- **Dark/Light theme:** Switch between themes dynamically  
- **React hooks:** `useState`, `useEffect`, and context for state management  
- **Chakra UI:** Pre-built components for fast UI development  
- Connecting frontend to backend API  

---


## 💡 Learning Outcomes

- Full MERN stack integration (MongoDB → Express → React → Node.js)  
- Component-based architecture and props/state management  
- Handling forms, validation, and API requests  
- Deployment of a full-stack app with a live link  
- Chakra UI and theme toggling
